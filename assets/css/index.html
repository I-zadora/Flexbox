Justify-content

A propriedade de conteúdo de justificativa do CSS define como o navegador distribui

espaço entre e ao redor de itens de conteúdo ao longo do eixo principal de um contêiner

flex e o eixo inline de um contêiner de grade.

Align-items

A propriedade CSS align-items estabelece o valor align-self em todos filhos diretos como um grupo.

A propriedade align-self estabelece o alinhamento de um certo item dentro do bloco que o contém

Flex-direction

definindo o eix

A propriedade CSS flex-direction define como os itens flexíveis são colocados no contêiner flexível

definindo o eixo principal e a direção (normal ou invertido).
